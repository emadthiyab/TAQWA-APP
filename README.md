# نظام متابعة أداء فندق دار التقوى

نظام متكامل لمتابعة أداء الموظفين والإدارات في فندق دار التقوى.

## المميزات

### واجهة الإدارة
- إدارة المستخدمين والصلاحيات
- إدارة الهيكل التنظيمي
- إدارة مؤشرات الأداء (KPIs)
- إدارة المشاريع
- إدارة الموظفين
- إدارة البلاغات

### واجهة المستخدمين
- متابعة المؤشرات الشخصية
- متابعة المشاريع
- تقييم الموظفين
- تسجيل البلاغات
- التقارير والمؤشرات
